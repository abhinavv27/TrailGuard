# TrailGuard AI — Reference Repositories

TrailGuard AI is original work. The following repositories provided **conceptual inspiration** for specific aspects of the platform. No code was copied; only ideas, patterns, and architectural approaches were studied.

---

## FinSentry

**Transaction Ingestion, Anomaly Detection, Graph Analytics**

- **URL**: https://github.com/diogomatoschaves/finsentry
- **Inspiration**: Data pipeline patterns for transaction ingestion, schema design for financial datasets, anomaly detection approaches for banking data.
- **Applied**: Transaction validation patterns, account-centric data model.

## Money Flow Visualization

**Fund-Flow Tree Interactions**

- **URL**: https://github.com/ocademy-ai/machine-learning
- **Inspiration**: Interactive visualization techniques for tracing money movement between accounts.
- **Applied**: D3.js force-directed graph approach for the Money Flow visualizer.

## Agentic AI Financial Crime Detection

**Workflow Orchestration**

- **URL**: https://github.com/elys/agentic-ai-financial-crime-detection
- **Inspiration**: Multi-stage detection pipeline architecture and alert triage workflows.
- **Applied**: Pipeline architecture for the detection engine (ingest → detect → analyze → report).

## Emergent-Code

**Engineering Quality Philosophy**

- **URL**: https://github.com/diogomatoschaves/emergent-code
- **Inspiration**: Testing philosophy, code organization, documentation-first approach.
- **Applied**: Test-driven approach to detection engine, structured documentation.

## Flue

**Agent Architecture Concepts**

- **URL**: https://github.com/FlueAgent/flue
- **Inspiration**: Agent-based workflow concepts for case management orchestration.
- **Applied**: Investigation state machine design (draft → open → under_review → closed).

## CORAL

**Development Orchestration Principles**

- **URL**: https://github.com/VaibhaviPatel105/CORAL
- **Inspiration**: Development workflow orchestration and multi-service coordination.
- **Applied**: Docker Compose service orchestration, development workflow scripts.
