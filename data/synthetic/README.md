# Synthetic Dataset

Generated by `data/generators/generate_synthetic_data.py`.

## Files

- `sample_transactions.csv` — 2500 normal + ~60 scenario transactions
- `sample_transactions.xlsx` — same data in Excel format

## Scenarios Planted

| Scenario | Label | Description | Count |
|---|---|---|---|
| Mule Ring | `SCENARIO_A` | 12 victims -> 1 mule -> 4 exits, rapid forwarding | 24 |
| Layering | `SCENARIO_B` | 4-hop layering chain ending in CH | 4 |
| Circular Flow | `SCENARIO_C` | 3-account cycle returning funds to origin | 3 |
| Structuring | `SCENARIO_D` | 15 deposits just under 10k via ATM | 15 |
| New Account HV | `SCENARIO_E` | 2-day-old account sending 20 large wires | 20 |

## Seeded Anomalies

- **Mule account** `MULE-AX7` created 14 days before activity
- **New account** `NEW-HV-01` aged 2 days with high outbound velocity
- **Structuring** amounts range \$7k–\$9,999 (just under reporting threshold)
- **Layering** crosses 3 countries (GB -> DE -> CH)
- **Circular** cycle completes back to origin within same day

## Usage

```bash
pip install openpyxl   # optional, for .xlsx output
python data/generators/generate_synthetic_data.py
```
